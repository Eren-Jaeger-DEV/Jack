# ❌ TicTacToe Stats

The `TicTacToeStats` model stores individual performance data for the bot's Tic-Tac-Toe minigame.

## 📋 Schema Fields

### `userId` (String, Unique)
The Discord ID of the player.

### 📊 Performance Metrics
- `wins`: Total games won.
- `losses`: Total games lost.
- `draws`: Total games ending in a tie.

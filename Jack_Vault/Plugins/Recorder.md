# 🎙️ Recorder

An advanced voice-recording system inspired by Craig, designed for capturing high-quality multi-track audio specifically from Discord voice channels.

## 📋 Features
- **Multi-Track**: Captures each user as a separate audio stream.
- **Auto-Join**: Can be configured to join specific channels automatically.
- **Privacy First**: Notifies all users when recording starts.

## 📂 Data Storage
- Local cache for temporary processing.
- Remote upload options available in certain configurations.

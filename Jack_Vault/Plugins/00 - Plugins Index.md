# 🧩 Plugins Index

Jack Bot is built on a highly modular architecture. Each feature is a standalone plugin located in the `plugins/` directory.

## 📋 Full Plugin List

### Core & Management
- [[System-Admin#Admin|Admin]]: Bot instance administration.
- [[System-Admin#Audit|Audit]]: Logs system and user actions.
- [[System-Admin#Logger|Logger]]: Centralized logging orchestration.
- [[System-Admin#Prefix|Prefix]]: Custom prefix management.

### Social & Members
- [[Economy-Social#Greeting|Greeting]]: Welcome and leave messages.
- [[Economy-Social#Leveling|Leveling]]: XP and leveling system.
- [[Member-Classification|Member-Classification]]: Role-based member sorting.
- [[Foster-Program|Foster-Program]]: Mentor/Rookie mentorship system.
- [[Utility-Misc#InviteTrackerAdvanced|InviteTrackerAdvanced]]: Advanced invite tracking and rewards.

### Moderation & Utility
- [[Moderation|Moderation]]: Disciplinary commands (Kick, Ban, Nickname).
- [[Support_Tickets|Tickets]]: Support ticket system.
- [[TempVC|TempVC]]: Personal voice channel management.
- [[Utility-Misc#Utility|Utility]]: General help and information tools.
- [[System-Admin#Triggers|Triggers]]: Keyword-based auto-responses.

### Games, Fun & Economy
- [[Utility-Misc#Fun|Fun]]: Action commands and entertainment.
- [[Games-Clan-Reference|Games]]: TTT, Minigames, and Leaderboards.
- [[Market|Market]]: Virtual marketplace.
- [[Economy-Social#Card-System (Packs, Database, Exchange)|Packs]]: Card/Item packs.
- [[Economy-Social#Card-System (Packs, Database, Exchange)|Card-Database]]: Inventory management.

### Competitive
- [[Clan-Battle|Clan-Battle]]: Clan wars and points.
- [[Intra-Match|Intra-Match]]: Internal clan scrims.
- [[Seasonal-Synergy|Seasonal-Synergy]]: Seasonal activity tracking.

### Server & Stats
- [[Server-Overview|Server-Overview]]: Interactive control panels.
- [[Server-Overview#Server-Stats|Server-Stats]]: Live data tracking.
- [[Utility-Misc#Celestial-HUD|Celestial-HUD]]: Premium UI overlays.
- [[Recorder|Recorder]]: Voice/Event recording tools.

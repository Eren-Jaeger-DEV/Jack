# ⚡ Command Reference: Games & Clans

Detailed usage for the bot's competitive and entertainment modules.

---

## 🎮 Game Commands
*Engagement-focused minigames and betting systems.*

| Command | Usage | Description |
| --- | --- | --- |
| `ttt` | `/ttt @user` | Challenge a member to Tic-Tac-Toe. |
| `tttlb` | `/tttlb` | View the global TTT leaderboard. |
| `coinflip` | `j coinflip [h/t]`| Bet currency on a coin toss. |
| `blackjack`| `j bj [amount]` | Play a round of Blackjack with the bot. |

---

## ⚔️ Clan Commands
*Registration and management for competitive play.*

| Command | Usage | Description |
| --- | --- | --- |
| `register`| `/register` | Link your Discord account for competitive matches. |
| `clan` | `/clan list` | View all registered clans. |
| `battle` | `/battle start` | Open registration for a Clan Battle. |
| `synergy` | `/synergy` | View your current seasonal synergy points. |

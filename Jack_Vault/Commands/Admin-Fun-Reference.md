# ⚡ Command Reference: Admin & Fun

This section provides detailed usage and permission requirements for the bot's core management and entertainment commands.

---

## 🛡️ Admin Commands
*High-level tools for bot and server maintenance.*

> [!CAUTION] **Bypass Warning**
> These commands typically require `Administrator` permissions or being listed in the `OWNER_IDS` array in `persona.js`.

| Command | Usage | Description |
| --- | --- | --- |
| `eval` | `j eval [code]` | Execute JavaScript code directly. |
| `reload` | `j reload [plugin]` | Hot-reload a specific plugin's logic. |
| `stats` | `/stats` | View bot performance and host metrics. |
| `shutdown` | `j shutdown` | Safely terminate the bot instance. |

---

## 🌈 Fun Commands
*Interactive and social actions to drive engagement.*

| Command | Usage | Description |
| --- | --- | --- |
| `hug` | `/hug @user` | Send a hug emotional action. |
| `slap` | `j slap @user` | Slap another user in chat. |
| `meme` | `/meme` | Generate a random meme. |
| `8ball` | `j 8ball [question]` | Ask the magic 8-ball a question. |

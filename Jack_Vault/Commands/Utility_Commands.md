# ⚡ Command Reference: Utility

This section provides reference for the bot's general utility and information tools. Most commands are accessible to all users.

---

## 🛠️ General Utility
*Information access, personal tools, and bot diagnostics.*

| Command | Usage | Type | Description |
| --- | --- | --- | --- |
| `help` | `/help` | Slash | View the interactive help menu with all available commands. |
| `ping` | `/ping` | Slash | Check bot latency and API ping. |
| `poll` | `/poll` | Slash | Create an interactive poll context. |
| `memberlist` | `/memberlist` | Slash | List all members with their roles. |
| `afk` | `j afk [status]` | Prefix | Set yourself as AFK. |
| `avatar` | `j avatar @user` | Prefix | Show a user's avatar image. |
| `serverinfo` | `j serverinfo` | Prefix | Show information about the current server. |
| `userinfo` | `j userinfo @user` | Prefix | Get detailed information about a target user. |
| `steal` | `j steal` (Reply)| Prefix | Steal an emoji or sticker by replying to a message and store it globally. |
| `prefixtest` | `j prefixtest` | Prefix | Test the current prefix system configuration. |
